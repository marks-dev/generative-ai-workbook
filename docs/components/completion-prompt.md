# Component Design: Completion Prompt (完了誘導ツールチップ)

## 用途
完了誘導ツールチップ（Completion Prompt）は、学習者が教材を未完了の状態でページ最下部まで読み進めた際に、ヘッダーの「完了にする」ボタンへの注目を促すための動的な吹き出しUIです。完了操作忘れを防ぎ、学習状況の確実な記録をサポートします。

## 利用画面
- 教材詳細ページ（未完了状態でスクロールが末尾に到達したときのみ動的に生成・表示）

## 動作・トリガー仕様
- **表示トリガー**:
  - スクリプトがスクロール位置を監視し、コンテンツ末尾（あるいは関連教材カード付近）に到達したことを検知。
  - 同時に、LocalStorageの進捗情報を確認し、現在の教材が「未完了」である場合のみトリガー。
- **表示位置**:
  - ヘッダー内の「完了にする」ボタン（Context Areaの右端）の直下に吹き出し形式で出現します。
- **消滅条件**:
  - 出現から3秒経過すると、自動的にフェードアウトし、DOMから削除または非表示になります（学習の邪魔にならないための配慮）。
  - または、ツールチップを直接クリック、もしくは完了ボタンをクリックした場合も即座に消滅します。

## デザイン仕様
- **グラスモーフィズム**:
  - 背景色: ティール半透明（`rgba(47, 111, 115, 0.88)`）
  - 背景ぼかし: `backdrop-filter: blur(8px)` または `blur(10px)`
  - 文字色: 白（読みやすさを確保）
  - 微細なドロップシャドウを適用し、コンテンツの手前に浮かび上がっているように表現します。
