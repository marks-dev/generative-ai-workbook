# Component Design: Related Content Card

## 用途
関連コンテンツカード（Related Content Card）は、教材詳細ページの最下部に配置され、現在学習しているテーマに関連する別の教材（座学からハンズオン、あるいはその逆）への相互移動を促すためのUI部品です。関連知識の補完や実践への移行をスムーズにします。

## 利用画面
- 教材詳細ページ（メインコンテンツ最下部）

## 構成・動作仕様
- **スタック配置**:
  - ページ最下部に、縦方向のスタック（縦積みリスト）形式で配置します。
- **動的なコンテンツ切替**:
  - 閲覧中の教材のディレクトリ構造に応じて、表示ラベルやアプローズを動的に切り替えます。
    - **`src/reading/` 配下の教材**: 「関連するハンズオン（🛠️）」という見出しで、実践用の教材へのリンクを表示。
    - **`src/hands-on/` 配下の教材**: 「関連する読み物（📖）」という見出しで、理論背景を学ぶ教材へのリンクを表示。
- **表示スタイル**:
  - タイトルとリンク、簡単な補足テキストを含み、周囲の本文と区別するための軽い枠線または微細な背景色を持ちます。
