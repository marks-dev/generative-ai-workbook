# Component Design: Learning Card

## 用途
ラーニングカード（Learning Card）は、カテゴリページにおいて、各教材を一覧表示するためのカード型UI部品です。教材の概要を簡潔に示し、学習者が興味のある教材を選択しやすくするとともに、進捗状況を視覚的に伝えます。

## 利用画面
- カテゴリページ
- トップページ（ルート入口タイルのバリエーションとして一部流用）

## 構成・レイアウト
カード全体はレスポンシブなグリッドまたはフレックスボックスを用いて、以下の3カラム構成を基本とします。

1. **左カラム (アイコン)**: 教材の性質を示す視覚的アイコン（読み物は 📖、ハンズオンは 🛠️ など）。
2. **中央カラム (メイン情報)**: 教材タイトルと、数行の簡潔な説明文。
3. **右カラム (完了バッジ)**: LocalStorageの学習進捗と同期し、完了時に表示される `✅` バッジ（インライン要素）。

## 動作・状態遷移
- **完了状態の表現**:
  - 対象の教材が学習完了されると、カード全体に `.is-completed` クラスが付与されます。
  - `.is-completed` 適用時、カードの枠線は柔らかなグリーン系に変化し、背景にはごく淡いグリーン（トーンを抑えた背景色）が適用され、一目で完了したことが分かります。
- **親カテゴリカードの連動**:
  - 親カテゴリを示すカードにおいては、そのカテゴリ内に存在するすべての子教材が完了した場合に、カード右端に ✅ バッジを表示します。
