# Component Design: Learning Progress Panel (学習状況管理パネル)

## 実装仕様 (Implementation Specification)
- **実装パターン**: パターン2（Nunjucks静的定義 ＋ 共通CSS ＋ JS状態連動）
- **主要関連ファイル**:
  - **HTML (テンプレート)**: [src/_includes/base.njk](../../src/_includes/base.njk)
  - **JavaScript (ダイアログ制御・入出力)**: [src/assets/js/learning-progress.js](../../src/assets/js/learning-progress.js)
  - **CSS (スタイル)**: [src/assets/css/site.css](../../src/assets/css/site.css)

## 用途
学習状況管理パネルは、学習者がLocalStorageに蓄積された学習履歴（完了した教材の一覧と件数）をダイアログ形式で確認し、データのインポート/エクスポートおよび初期化（リセット）を行うためのユーティリティUIです。

## 利用画面
- 全画面（ヘッダーの「学習状況管理」ボタンから呼び出し可能）

## 構成・レイアウト
ダイアログは画面右側からスライドインするサイドパネル形式で表示されます（モバイル環境では全画面表示）。
- **Backdrop (背景マスク)**: ダイアログの背後を暗くし、フォーカスを集中させます。
- **Header**: タイトル（「学習状況管理」）と「閉じる」ボタン。
- **Summary**: 完了済み教材の件数表示。
- **List**: 完了した教材のタイトルと完了日付を逆時系列（新しい順）で表示するスクロール可能な領域。
- **Actions (操作ボタン群)**:
  - **JSONエクスポート**: データをJSONファイルとしてダウンロード。
  - **JSONインポート**: JSONファイルを読み込みLocalStorageを上書き。
  - **全状況をリセット**: LocalStorageのデータを初期状態に戻す危険なアクション。
- **Message Area**: 操作結果（「インポートしました」等）を通知するテキストエリア。

## 動作・アクセシビリティ（A11y）仕様
- **表示・非表示のトグル**:
  - パネルの開閉状態に応じて、`<html>` に `.is-learning-progress-open` クラスを付与し、パネル本体の `hidden` 属性を切り替えます。
- **アクセシビリティ考慮**:
  - ダイアログには `role="dialog"`, `aria-modal="true"`, `aria-labelledby` を付与し、スクリーンリーダーに対応します。
  - パネル展開時には、キーボードフォーカスをパネル内の「閉じる」ボタンへ自動で移動させます。
  - `Escape` キー押下、または Backdrop 領域のクリックによって即座にダイアログを閉じるキーボード操作・ジェスチャに対応します。
  - メッセージエリアには `aria-live="polite"` を指定し、インポート/エクスポート結果が音声で動的に通知されるようにします。
- **データ操作時の処理**:
  - **インポート**: JSON構造の形式チェックを行い、不正なデータの場合は警告メッセージを表示して処理を中断します。
  - **リセット**: ユーザーに確認用ダイアログ（`window.confirm`）を表示し、明確な承認がない限り処理を実行しません。
