# Component Design: Navigation Tile

## 用途
ナビゲーションタイル（Navigation Tile）は、トップページにおいて学習者が主要な学習ルート（「読んで学ぶ」または「触って学ぶ」）を選択するための入り口となるカード型UI部品です。複雑な説明を廃し、直感的にルートを選択できるシンプルな構成を持ちます。

## 利用画面
- トップページ

## 構成・デザイン
- **要素定義 (HTMLクラス `top-route-card`)**:
  - タイル全体をアンカーリンク (`<a class="top-route-card">`) とし、クリック領域を最大化します。
  - 内部は絵文字アイコン (`<span class="top-route-card__icon">`) とルート名を示す見出し (`<h2>`) のみで構成します。
  - カード内に余計な説明テキストやサブボタンは配置しません。
- **インタラクション (CSS)**:
  - ホバー (`:hover`) およびフォーカス (`:focus-visible`) 時に、ボーダーがアクセントカラー（`--color-accent`）に変化し、背景がアクセントカラーの薄い配色（`--color-accent-muted`）へ変化します。
  - キーボード操作での移動に配慮し、`:focus-visible` によるフォーカス枠を表示します。
