# About Page Design

## 目的

「このサイトについて」ページは、本サイトの概要、提供背景、特徴、学習履歴の扱い、およびライセンス等について説明する画面です。
利用者がサイトの趣旨を正しく理解し、安心して学習を進められるようにすることを目的とします。

---

## 基本方針

- 画面のノイズを極力排し、読みやすさを最優先したテキスト主体のレイアウトとする。
- 「読んで学ぶ」「触って学ぶ」の相互リンクを推奨するコンセプト（本サイトの最大の特徴）を強調する。
- 個人情報保護・プライバシーに配慮し、サーバーに学習履歴を保存しない設計（LocalStorage完結）であることを明記する。
- オープンソース（MIT License）としての公開方針を示す。

## 各セクションの詳細

### 1. 開発の背景と目的
- 実務でのAIスキル修得の必要性と、「何から学べばいいか分からない」といった課題に対する自律学習ポータルとしての価値を解説します。

### 2. 本サイトの特徴
- インプットとしての「読んで学ぶ」とアウトプットとしての「触って学ぶ」の2本柱を説明。
- **相互リンク推奨（TIP）**:
  Notion風のAlertスタイル（`[!TIP]`）を用いて、座学とハンズオンを相互に行き来する効果的な学習手順を促します。

### 3. 学習状況の管理について
- 利用者のプライバシー保護のため、サーバーへのデータ送信は行わず、ブラウザのLocalStorageで完結する仕組みであることを分かりやすく解説。
- データのバックアップや端末間移行手段として、JSONエクスポート・インポート機能が提供されていることを案内します。
