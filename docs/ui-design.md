

# UI Design

## 目的

本ファイルはUI設計方針を管理する。

レイアウト詳細やコンポーネント詳細ではなく、サイト全体のUI設計思想を記録する。

---

## 対象デバイス

優先対象:

- デスクトップPC
- ノートPC

対応対象:

- タブレット
- スマートフォン

---

## 基本方針

- PCファーストで設計する
- レスポンシブ対応を行う
- スマートフォンでの閲覧を可能とする
- スマートフォンでの学習も考慮する
- ただし全コンテンツのスマートフォン完遂は前提としない

例:

- AIコーディング
- MCP
- RAG

などはPC利用を前提としてもよい。

---

## ナビゲーション方針

PC:

- ヘッダー
- サイドバー
- メインコンテンツ

スマートフォン:

- ヘッダー
- ハンバーガーメニュー
- メインコンテンツ

教材サイトとしての学習導線を重視する。
