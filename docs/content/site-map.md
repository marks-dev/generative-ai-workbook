

# Site Map

## サイトの目的

株式会社MARKSが社員向けに作成した生成AI自習用ワークブック。

現在は MIT License で公開しており、利用者は自由に利用・改変できる。

本サイトは単なる教材置き場ではなく、生成AIを継続的に学ぶための学習ポータルとして提供する。

---

## 学習コンセプト

利用者によって学びやすい順番は異なる。

- 座学から学びたい人
- 手を動かしながら学びたい人

どちらにも対応するため、以下の2つの入口を提供する。

- 読んで学ぶ
- 触って学ぶ

また、すべての教材は可能な限り相互リンクを設置し、以下の導線を実現する。

```text
読む ⇄ 触る
```

### 推奨学習順序

1. はじめての生成AI
   - 無料プランで学習可能

2. 仕事で活用する
   - 有料プラン推奨

3. AIと仕組みを作る
   - 開発向け機能や高度なモデルの利用を推奨

## 学習状況管理

GitHub Pagesのためサーバー保存は行わない。

ブラウザのローカルストレージを利用して以下を管理する。

- 未実施
- 完了

初期実装では、上記2状態のみ管理する。

学習状況管理は独立ページを作成せず、Headerの「学習状況管理」から利用できるようにする。

学習状況をJSON形式でエクスポート・インポートできるようにする。
