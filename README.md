# 生成AIワークブック

生成AIを体系的に学ぶための自習式教材です。

学習者が自分のペースで学べることを重視し、小さなステップで理解を積み上げられる構成を目指しています。

## 対象者

- 生成AIをこれから学びたい方
- ChatGPTや各種生成AIサービスに興味がある方
- 自習形式で学習を進めたい方
- 教材やサンプルを作りながら学びたい方

## 特徴

- 自習形式で学べる
- 初心者向けのステップ構成
- GitHub Pagesで公開可能
- 静的サイトとして運用可能
- 実践しながら学べるサンプルを含む

## 学習者向け

1. 教材を順番に読み進める
2. サンプルを実際に試す
3. 演習課題に取り組む
4. 必要に応じて復習する

## 開発者向け

この教材は静的サイトとして公開することを想定しています。

### 開発方針

- シンプルで理解しやすい構成を優先する
- 初心者が迷わない導線を意識する
- 小さな単位で学習できる教材を作る
- GitHub Pagesで公開できる構成を維持する

## 公開

GitHub Pagesでの公開を想定しています。
